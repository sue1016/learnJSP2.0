###  jsp简介   
JSP全名为Java Server Pages,其根本是一个简化的Servlet设计，**他实现了在JAVA当中使用HTML标签**。  
JSP是一种动态网页技术标准，也是JAVAEE的标准。   
JSP与Selvlet一样，是在**服务器端执行**的。
